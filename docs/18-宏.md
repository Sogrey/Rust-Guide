# Rust 宏

宏是 Rust 中强大的元编程工具，允许编写生成代码的代码。

## 宏 vs 函数

| 特性 | 宏 | 函数 |
|------|-----|------|
| 参数数量 | 可变 | 固定 |
| 类型检查 | 展开后检查 | 调用时检查 |
| 执行时机 | 编译时 | 运行时 |
| 性能 | 无运行时开销 | 函数调用开销 |
| 复杂度 | 高 | 低 |

## 声明宏（macro_rules!）

### 基本语法

```rust
// 定义简单宏
macro_rules! say_hello {
    () => {
        println!("Hello!");
    };
}

fn main() {
    say_hello!();
}
```

### 带参数的宏

```rust
macro_rules! create_function {
    ($func_name:ident) => {
        fn $func_name() {
            println!("Function {:?} called", stringify!($func_name));
        }
    };
}

fn main() {
    create_function!(foo);
    create_function!(bar);
    
    foo();
    bar();
}
```

### 参数类型

```rust
macro_rules! demo {
    // 表达式
    ($e:expr) => {
        println!("Expression: {}", $e);
    };
    
    // 标识符
    ($i:ident) => {
        println!("Identifier: {}", stringify!($i));
    };
    
    // 类型
    ($t:ty) => {
        println!("Type: {}", stringify!($t));
    };
    
    // 字面量
    ($l:literal) => {
        println!("Literal: {}", $l);
    };
    
    // 语句
    ($s:stmt) => {
        println!("Statement: {}", stringify!($s));
    };
    
    // 模式
    ($p:pat) => {
        println!("Pattern: {}", stringify!($p));
    };
    
    // 路径
    ($path:path) => {
        println!("Path: {}", stringify!($path));
    };
    
    // 可重复
    ($($x:expr),*) => {
        $(println!("Repeat: {}", $x);)*
    };
}

fn main() {
    demo!(1 + 2);          // expr
    demo!(x);               // ident
    demo!(i32);             // ty
    demo!(42);              // literal
    demo!(let x = 1;);      // stmt
    demo!(Some(x));         // pat
    demo!(std::collections::HashMap);  // path
    demo!(1, 2, 3);         // repeat
}
```

### 可变参数

```rust
// 计算平均值
macro_rules! average {
    ($($x:expr),*) => {
        {
            let count = 0usize $(+ { let _ = $x; 1 })*;
            let sum = 0f64 $(+ ($x as f64))*;
            sum / count as f64
        }
    };
}

fn main() {
    println!("Average: {}", average!(1, 2, 3, 4, 5));
}

// 向量创建宏
macro_rules! my_vec {
    ($($x:expr),*) => {
        {
            let mut temp_vec = Vec::new();
            $(
                temp_vec.push($x);
            )*
            temp_vec
        }
    };
    
    ($($x:expr),+ ,) => {
        my_vec![$($x),+]
    };
}

fn main() {
    let v = my_vec![1, 2, 3];
    println!("{:?}", v);
}
```

### 重复模式

```rust
macro_rules! build_struct {
    ($name:ident { $($field:ident : $type:ty),* }) => {
        struct $name {
            $( $field : $type ),*
        }
    };
}

fn main() {
    build_struct!(Person {
        name: String,
        age: u32,
        active: bool
    });
    
    let person = Person {
        name: String::from("Alice"),
        age: 30,
        active: true,
    };
    
    println!("{}: {}", person.name, person.age);
}
```

### 常用内置宏

```rust
fn main() {
    // println! - 格式化打印
    println!("Hello, {}!", "World");
    
    // format! - 格式化字符串
    let s = format!("Value: {}", 42);
    
    // vec! - 创建向量
    let v = vec![1, 2, 3];
    
    // panic! - 触发 panic
    // panic!("Something went wrong");
    
    // assert! - 断言
    assert!(2 + 2 == 4);
    
    // assert_eq! - 相等断言
    assert_eq!(2 + 2, 4);
    
    // assert_ne! - 不等断言
    assert_ne!(2 + 2, 5);
    
    // dbg! - 调试打印
    let x = 42;
    dbg!(x);
    
    // todo! - 未实现标记
    // fn unimplemented() { todo!() }
    
    // unimplemented! - 未实现
    // fn not_done() { unimplemented!() }
    
    // cfg! - 条件编译
    if cfg!(debug_assertions) {
        println!("Debug mode");
    }
    
    // env! - 环境变量
    // let path = env!("PATH");
    
    // concat! - 字符串连接
    let s = concat!("Hello", " ", "World");
    
    // stringify! - 转字符串
    let s = stringify!(1 + 2);
    
    // include_str! - 包含文件内容
    // let content = include_str!("data.txt");
    
    // include_bytes! - 包含文件字节
    // let bytes = include_bytes!("data.bin");
}
```

## 过程宏（Procedural Macros）

过程宏是 Rust 函数，接收代码作为输入，生成代码作为输出。

### 设置项目结构

```toml
# Cargo.toml
[package]
name = "my_macros"
version = "0.1.0"
edition = "2021"

[lib]
proc-macro = true

[dependencies]
syn = "2.0"
quote = "1.0"
proc-macro2 = "1.0"
```

### 派生宏（Derive Macros）

```rust
// lib.rs
use proc_macro::TokenStream;
use quote::quote;
use syn::{parse_macro_input, DeriveInput};

#[proc_macro_derive(HelloMacro)]
pub fn hello_macro_derive(input: TokenStream) -> TokenStream {
    let input = parse_macro_input!(input as DeriveInput);
    
    let name = &input.ident;
    let expanded = quote! {
        impl HelloMacro for #name {
            fn hello_macro() {
                println!("Hello, Macro! My name is {}!", stringify!(#name));
            }
        }
    };
    
    TokenStream::from(expanded)
}
```

使用派生宏：

```rust
// main.rs
use my_macros::HelloMacro;

trait HelloMacro {
    fn hello_macro();
}

#[derive(HelloMacro)]
struct Pancakes;

fn main() {
    Pancakes::hello_macro();
}
```

### 属性宏（Attribute Macros）

```rust
// lib.rs
use proc_macro::TokenStream;
use quote::quote;
use syn::{parse_macro_input, ItemFn};

#[proc_macro_attribute]
pub fn timed(_attr: TokenStream, item: TokenStream) -> TokenStream {
    let input = parse_macro_input!(item as ItemFn);
    let fn_name = &input.sig.ident;
    let fn_block = &input.block;
    
    let expanded = quote! {
        fn #fn_name() {
            let start = std::time::Instant::now();
            #fn_block
            println!("Function took {:?}", start.elapsed());
        }
    };
    
    TokenStream::from(expanded)
}
```

使用属性宏：

```rust
// main.rs
use my_macros::timed;

#[timed]
fn expensive_function() {
    std::thread::sleep(std::time::Duration::from_secs(1));
}

fn main() {
    expensive_function();
}
```

### 函数式宏（Function-like Macros）

```rust
// lib.rs
use proc_macro::TokenStream;
use quote::quote;
use syn::parse::{Parse, ParseStream};
use syn::{parse_macro_input, Ident, Token};

struct Args {
    name: Ident,
}

impl Parse for Args {
    fn parse(input: ParseStream) -> syn::Result<Self> {
        let name: Ident = input.parse()?;
        Ok(Args { name })
    }
}

#[proc_macro]
pub fn make_function(input: TokenStream) -> TokenStream {
    let Args { name } = parse_macro_input!(input as Args);
    
    let expanded = quote! {
        fn #name() {
            println!("Function {} called", stringify!(#name));
        }
    };
    
    TokenStream::from(expanded)
}
```

使用函数式宏：

```rust
// main.rs
use my_macros::make_function;

make_function!(my_func);

fn main() {
    my_func();
}
```

## 实用宏示例

### 单例模式宏

```rust
macro_rules! singleton {
    ($name:ident : $type:ty = $value:expr) => {
        static $name: std::sync::OnceLock<$type> = std::sync::OnceLock::new();
        
        fn $name() -> &'static $type {
            $name.get_or_init(|| $value)
        }
    };
}

fn main() {
    singleton!(CONFIG: String = String::from("config value"));
    
    println!("{}", CONFIG());
}
```

### 测试宏

```rust
macro_rules! test_cases {
    ($($name:ident: $input:expr => $expected:expr),*) => {
        $(
            #[test]
            fn $name() {
                assert_eq!($input, $expected);
            }
        )*
    };
}

test_cases! {
    test_add: 2 + 2 => 4
    test_mul: 3 * 3 => 9
    test_sub: 10 - 5 => 5
}
```

## 最佳实践

### 1. 优先使用函数

```rust
// ✅ 使用函数
fn add(a: i32, b: i32) -> i32 {
    a + b
}

// ❌ 避免不必要的宏
macro_rules! add {
    ($a:expr, $b:expr) => {
        $a + $b
    };
}
```

### 2. 文档化宏

```rust
/// 计算多个数的平均值
/// 
/// # Examples
/// 
/// ```
/// let avg = average!(1, 2, 3);
/// assert_eq!(avg, 2.0);
/// ```
macro_rules! average {
    // ...
}
```

### 3. 卫生性（Hygiene）

```rust
// ✅ 卫生的宏
macro_rules! safe_macro {
    ($x:expr) => {
        {
            let temp = $x;
            temp * 2
        }
    };
}

// ❌ 不卫生的宏（可能引起命名冲突）
macro_rules! unsafe_macro {
    ($x:expr) => {
        let result = $x * 2;
        result
    };
}
```
