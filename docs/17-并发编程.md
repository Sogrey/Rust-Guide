# Rust 并发编程

Rust 的并发编程以"无畏并发"(Fearless Concurrency)著称，通过所有权系统在编译时防止数据竞争。

## 线程

### 创建线程

```rust
use std::thread;
use std::time::Duration;

fn main() {
    // 创建新线程
    let handle = thread::spawn(|| {
        for i in 1..=5 {
            println!("子线程: {}", i);
            thread::sleep(Duration::from_millis(100));
        }
    });
    
    // 主线程
    for i in 1..=3 {
        println!("主线程: {}", i);
        thread::sleep(Duration::from_millis(100));
    }
    
    // 等待线程结束
    handle.join().unwrap();
}
```

### 使用 move 闭包

```rust
use std::thread;

fn main() {
    let message = String::from("Hello from thread");
    
    // 使用 move 将所有权转移到线程
    let handle = thread::spawn(move || {
        println!("{}", message);
    });
    
    handle.join().unwrap();
}
```

### 线程返回值

```rust
use std::thread;

fn main() {
    let handle = thread::spawn(|| {
        let result = 1 + 2;
        result * 2
    });
    
    let result = handle.join().unwrap();
    println!("Result: {}", result);
}
```

## 消息传递

### 通道(Channel)

```rust
use std::sync::mpsc;
use std::thread;

fn main() {
    // 创建通道
    let (tx, rx) = mpsc::channel();
    
    // 发送线程
    thread::spawn(move || {
        let messages = vec![
            String::from("Hi"),
            String::from("from"),
            String::from("thread"),
        ];
        
        for msg in messages {
            tx.send(msg).unwrap();
            thread::sleep(std::time::Duration::from_millis(100));
        }
    });
    
    // 接收消息
    for received in rx {
        println!("Received: {}", received);
    }
}
```

### 多生产者

```rust
use std::sync::mpsc;
use std::thread;

fn main() {
    let (tx, rx) = mpsc::channel();
    
    // 克隆发送端
    let tx1 = tx.clone();
    let tx2 = tx.clone();
    
    // 生产者 1
    thread::spawn(move || {
        tx1.send(String::from("Producer 1")).unwrap();
    });
    
    // 生产者 2
    thread::spawn(move || {
        tx2.send(String::from("Producer 2")).unwrap();
    });
    
    // 主线程也是生产者
    drop(tx);  // 释放原始 tx
    
    for received in rx {
        println!("Got: {}", received);
    }
}
```

### 同步通道

```rust
use std::sync::mpsc::sync_channel;
use std::thread;

fn main() {
    // 创建同步通道，缓冲区大小为 1
    let (tx, rx) = sync_channel(1);
    
    thread::spawn(move || {
        tx.send(1).unwrap();
        println!("Sent 1");
        
        tx.send(2).unwrap();  // 会阻塞直到第一个值被接收
        println!("Sent 2");
    });
    
    thread::sleep(std::time::Duration::from_secs(1));
    println!("Received: {}", rx.recv().unwrap());
    
    thread::sleep(std::time::Duration::from_secs(1));
    println!("Received: {}", rx.recv().unwrap());
}
```

## 共享状态

### Mutex(互斥锁)

```rust
use std::sync::{Arc, Mutex};
use std::thread;

fn main() {
    // 使用 Arc 和 Mutex 实现共享状态
    let counter = Arc::new(Mutex::new(0));
    let mut handles = vec![];
    
    for _ in 0..10 {
        let counter = Arc::clone(&counter);
        let handle = thread::spawn(move || {
            let mut num = counter.lock().unwrap();
            *num += 1;
        });
        handles.push(handle);
    }
    
    for handle in handles {
        handle.join().unwrap();
    }
    
    println!("Result: {}", *counter.lock().unwrap());
}
```

### RwLock(读写锁)

```rust
use std::sync::{Arc, RwLock};
use std::thread;

fn main() {
    let data = Arc::new(RwLock::new(vec![1, 2, 3]));
    let mut handles = vec![];
    
    // 写线程
    for i in 0..3 {
        let data = Arc::clone(&data);
        let handle = thread::spawn(move || {
            let mut w = data.write().unwrap();
            w.push(i);
        });
        handles.push(handle);
    }
    
    // 读线程
    for _ in 0..3 {
        let data = Arc::clone(&data);
        let handle = thread::spawn(move || {
            let r = data.read().unwrap();
            println!("Read: {:?}", *r);
        });
        handles.push(handle);
    }
    
    for handle in handles {
        handle.join().unwrap();
    }
    
    println!("Final: {:?}", *data.read().unwrap());
}
```

### Atomic(原子类型)

```rust
use std::sync::atomic::{AtomicUsize, Ordering};
use std::sync::Arc;
use std::thread;

fn main() {
    let counter = Arc::new(AtomicUsize::new(0));
    let mut handles = vec![];
    
    for _ in 0..10 {
        let counter = Arc::clone(&counter);
        let handle = thread::spawn(move || {
            counter.fetch_add(1, Ordering::SeqCst);
        });
        handles.push(handle);
    }
    
    for handle in handles {
        handle.join().unwrap();
    }
    
    println!("Result: {}", counter.load(Ordering::SeqCst));
}
```

## Send 和 Sync Trait

### Send Trait

`Send` 标记 trait 表示类型可以在线程间转移所有权：

```rust
// 大多数类型都是 Send
// Rc<T> 不是 Send（不是线程安全）
// Arc<T> 是 Send（线程安全）

use std::rc::Rc;
use std::sync::Arc;

fn main() {
    // ❌ 不能在线程间传递
    // let rc = Rc::new(5);
    // thread::spawn(move || {
    //     println!("{}", *rc);
    // });
    
    // ✅ 使用 Arc
    let arc = Arc::new(5);
    let arc_clone = Arc::clone(&arc);
    thread::spawn(move || {
        println!("{}", *arc_clone);
    }).join().unwrap();
}
```

### Sync Trait

`Sync` 标记 trait 表示类型可以在线程间共享引用：

```rust
// 如果 &T 是 Send，则 T 是 Sync
// Mutex<T> 是 Sync
// RefCell<T> 不是 Sync

use std::cell::RefCell;
use std::sync::Mutex;

fn main() {
    // ❌ RefCell 不是 Sync
    // let refcell = RefCell::new(5);
    // thread::spawn(move || {
    //     *refcell.borrow_mut() = 10;
    // });
    
    // ✅ 使用 Mutex
    let mutex = Mutex::new(5);
    thread::spawn(move || {
        *mutex.lock().unwrap() = 10;
    }).join().unwrap();
}
```

## Barrier

```rust
use std::sync::{Arc, Barrier};
use std::thread;

fn main() {
    let mut handles = vec![];
    let barrier = Arc::new(Barrier::new(3));
    
    for i in 0..3 {
        let barrier = Arc::clone(&barrier);
        let handle = thread::spawn(move || {
            println!("Thread {} before barrier", i);
            barrier.wait();
            println!("Thread {} after barrier", i);
        });
        handles.push(handle);
    }
    
    for handle in handles {
        handle.join().unwrap();
    }
}
```

## Condvar(条件变量)

```rust
use std::sync::{Arc, Mutex, Condvar};
use std::thread;

fn main() {
    let pair = Arc::new((Mutex::new(false), Condvar::new()));
    let pair2 = Arc::clone(&pair);
    
    // 等待线程
    let handle = thread::spawn(move || {
        let (lock, cvar) = &*pair2;
        let mut started = lock.lock().unwrap();
        while !*started {
            started = cvar.wait(started).unwrap();
        }
        println!("Thread started!");
    });
    
    // 主线程触发
    thread::sleep(std::time::Duration::from_secs(1));
    let (lock, cvar) = &*pair;
    let mut started = lock.lock().unwrap();
    *started = true;
    cvar.notify_one();
    
    handle.join().unwrap();
}
```

## 线程池

### 简单线程池

```rust
use std::sync::{Arc, Mutex};
use std::thread;
use std::sync::mpsc;

type Job = Box<dyn FnOnce() + Send + 'static>;

struct ThreadPool {
    workers: Vec<thread::JoinHandle<()>>,
    sender: mpsc::Sender<Job>,
}

impl ThreadPool {
    fn new(size: usize) -> Self {
        let (sender, receiver) = mpsc::channel();
        let receiver = Arc::new(Mutex::new(receiver));
        
        let mut workers = Vec::with_capacity(size);
        
        for _ in 0..size {
            let receiver = Arc::clone(&receiver);
            let worker = thread::spawn(move || loop {
                let job = receiver.lock().unwrap().recv().unwrap();
                job();
            });
            workers.push(worker);
        }
        
        ThreadPool { workers, sender }
    }
    
    fn execute<F>(&self, f: F)
    where
        F: FnOnce() + Send + 'static,
    {
        let job = Box::new(f);
        self.sender.send(job).unwrap();
    }
}

fn main() {
    let pool = ThreadPool::new(4);
    
    for i in 0..8 {
        pool.execute(move || {
            println!("Task {} executing", i);
        });
    }
    
    thread::sleep(std::time::Duration::from_secs(1));
}
```

## 最佳实践

### 1. 避免数据竞争

```rust
// ✅ 使用 Mutex 保护共享数据
use std::sync::{Arc, Mutex};

let data = Arc::new(Mutex::new(vec![]));
```

### 2. 减少锁的持有时间

```rust
use std::sync::Mutex;

fn bad_example(mutex: &Mutex<Vec<i32>>) {
    let mut data = mutex.lock().unwrap();
    // 长时间持有锁
    process_data(&mut data);
}

fn good_example(mutex: &Mutex<Vec<i32>>) {
    let data = {
        let mut data = mutex.lock().unwrap();
        data.clone()  // 快速复制
    };  // 释放锁
    process_data(&mut data);  // 在锁外处理
}

fn process_data(data: &mut Vec<i32>) {
    // ...
}
```

### 3. 使用消息传递而非共享状态

```rust
// ✅ 推荐：消息传递
let (tx, rx) = mpsc::channel();

// ✅ 在需要时使用共享状态
let shared = Arc::new(Mutex::new(data));
```
