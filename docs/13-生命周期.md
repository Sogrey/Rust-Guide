# Rust 教程 - 生命周期详解

## 本章概述

生命周期确保引用始终有效。本章详细介绍生命周期注解、省略规则和高级用法。

## 学习目标

- 理解生命周期概念
- 掌握生命周期注解
- 理解生命周期省略规则
- 理解 'static 生命周期
- 掌握生命周期约束

## 1. 生命周期概念

生命周期是引用有效的范围：

```rust
{
    let r;                // ---------+-- 'a
                          //          |
    {                     //          |
        let x = 5;        // -+-- 'b  |
        r = &x;           //  |       |
    }                     // -+       |
                          //          |
    // println!("{}", r); // 错误！x 已离开作用域
}                         // ---------+
```

## 2. 生命周期注解

```rust
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() {
        x
    } else {
        y
    }
}
```

## 3. 结构体中的生命周期

```rust
struct ImportantExcerpt<'a> {
    part: &'a str,
}
```

## 4. 生命周期省略规则

编译器使用三条规则自动推断：

1. 每个引用参数获得一个生命周期
2. 只有一个输入生命周期时，赋予所有输出生命周期
3. 有 `&self` 或 `&mut self` 时，self 的生命周期赋予所有输出生命周期

## 5. 'static 生命周期

整个程序运行期间都有效：

```rust
let s: &'static str = "我有 static 生命周期";
```

## 6. 生命周期约束

```rust
fn longest_with_announcement<'a, T>(
    x: &'a str,
    y: &'a str,
    ann: T,
) -> &'a str
where
    T: std::fmt::Display,
{
    println!("公告: {}", ann);
    if x.len() > y.len() { x } else { y }
}
```
