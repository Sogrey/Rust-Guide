# Rust 教程 - 结构体详解

## 本章概述

结构体是创建自定义数据类型的核心方式。本章详细介绍结构体的定义、实例化、方法和trait实现。

## 学习目标

- 掌握结构体的定义和实例化
- 理解元组结构体和单元结构体
- 掌握方法的定义
- 学会使用关联函数
- 理解结构体的可见性

## 1. 定义结构体

### 1.1 基本语法

```rust
struct User {
    username: String,
    email: String,
    active: bool,
    sign_in_count: u64,
}
```

### 1.2 创建实例

```rust
let user1 = User {
    email: String::from("user@example.com"),
    username: String::from("user123"),
    active: true,
    sign_in_count: 1,
};
```

### 1.3 访问字段

```rust
println!("用户名: {}", user1.username);
println!("邮箱: {}", user1.email);
```

## 2. 可变结构体

整个实例必须是可变的：

```rust
let mut user2 = User {
    email: String::from("another@example.com"),
    username: String::from("another"),
    active: true,
    sign_in_count: 1,
};

user2.email = String::from("new_email@example.com");
```

**注意**：不能只将某些字段标记为可变。

## 3. 结构体更新语法

使用其他实例创建新实例：

```rust
let user3 = User {
    email: String::from("new@example.com"),
    ..user1  // 其余字段来自 user1
};
```

**注意**：`..user1` 必须放在最后。

## 4. 元组结构体

没有字段名的结构体：

```rust
struct Color(i32, i32, i32);
struct Point(i32, i32, i32);

let black = Color(0, 0, 0);
let origin = Point(0, 0, 0);
```

## 5. 单元结构体

没有字段的结构体：

```rust
struct AlwaysEqual;
let subject = AlwaysEqual;
```

## 6. 方法

方法定义在 `impl` 块中：

```rust
impl Rectangle {
    fn area(&self) -> u32 {
        self.width * self.height
    }
}
```

## 7. 关联函数

没有 `self` 参数的函数：

```rust
impl Rectangle {
    fn square(size: u32) -> Rectangle {
        Rectangle { width: size, height: size }
    }
}
```
