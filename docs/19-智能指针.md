# Rust 智能指针

智能指针是指除了指向数据外，还拥有额外元数据和功能的指针。Rust 标准库提供了多种智能指针类型。

## Box<T> - 堆分配

### 基本使用

```rust
fn main() {
    // 在堆上分配值
    let b = Box::new(5);
    println!("b = {}", b);
    
    // 递归类型
    #[derive(Debug)]
    enum List {
        Cons(i32, Box<List>),
        Nil,
    }
    
    let list = List::Cons(1, Box::new(List::Cons(2, Box::new(List::Nil))));
    println!("{:?}", list);
}
```

### 大型数据转移

```rust
fn main() {
    // 避免栈溢出
    let large_data = Box::new([0u8; 1_000_000]);
    println!("First byte: {}", large_data[0]);
}
```

## Rc<T> - 引用计数

### 共享所有权（单线程）

```rust
use std::rc::Rc;

fn main() {
    let a = Rc::new(5);
    
    // 增加引用计数
    let b = Rc::clone(&a);
    let c = Rc::clone(&a);
    
    println!("Reference count: {}", Rc::strong_count(&a));  // 3
    println!("Values: a={}, b={}, c={}", a, b, c);
}
```

### 共享数据结构

```rust
use std::rc::Rc;

#[derive(Debug)]
struct Node {
    value: i32,
    children: Vec<Rc<Node>>,
}

fn main() {
    let leaf = Rc::new(Node {
        value: 3,
        children: vec![],
    });
    
    let branch = Node {
        value: 5,
        children: vec![Rc::clone(&leaf)],
    };
    
    println!("{:?}", branch);
}
```

## Arc<T> - 原子引用计数

### 线程安全的共享所有权

```rust
use std::sync::Arc;
use std::thread;

fn main() {
    let data = Arc::new(vec![1, 2, 3, 4, 5]);
    let mut handles = vec![];
    
    for _ in 0..3 {
        let data = Arc::clone(&data);
        let handle = thread::spawn(move || {
            println!("{:?}", *data);
        });
        handles.push(handle);
    }
    
    for handle in handles {
        handle.join().unwrap();
    }
}
```

## RefCell<T> - 内部可变性

### 运行时借用检查

```rust
use std::cell::RefCell;

fn main() {
    let data = RefCell::new(5);
    
    // 不可变借用
    println!("Value: {}", *data.borrow());
    
    // 可变借用
    *data.borrow_mut() += 1;
    println!("New value: {}", *data.borrow());
    
    // 运行时检查：同时有多个可变借用会 panic
    // let mut ref1 = data.borrow_mut();
    // let mut ref2 = data.borrow_mut();  // panic!
}
```

### 与 Rc 组合

```rust
use std::cell::RefCell;
use std::rc::Rc;

#[derive(Debug)]
struct Node {
    value: i32,
    children: RefCell<Vec<Rc<Node>>>,
}

fn main() {
    let leaf = Rc::new(Node {
        value: 3,
        children: RefCell::new(vec![]),
    });
    
    let branch = Rc::new(Node {
        value: 5,
        children: RefCell::new(vec![Rc::clone(&leaf)]),
    });
    
    // 修改叶子节点
    leaf.children.borrow_mut().push(Rc::clone(&branch));
}
```

## Mutex<T> - 互斥锁

```rust
use std::sync::{Arc, Mutex};
use std::thread;

fn main() {
    let counter = Arc::new(Mutex::new(0));
    let mut handles = vec![];
    
    for _ in 0..10 {
        let counter = Arc::clone(&counter);
        let handle = thread::spawn(move || {
            let mut num = counter.lock().unwrap();
            *num += 1;
        });
        handles.push(handle);
    }
    
    for handle in handles {
        handle.join().unwrap();
    }
    
    println!("Result: {}", *counter.lock().unwrap());
}
```

## RwLock<T> - 读写锁

```rust
use std::sync::{Arc, RwLock};
use std::thread;

fn main() {
    let data = Arc::new(RwLock::new(vec![1, 2, 3]));
    let mut handles = vec![];
    
    // 写线程
    {
        let data = Arc::clone(&data);
        let handle = thread::spawn(move || {
            let mut w = data.write().unwrap();
            w.push(4);
        });
        handles.push(handle);
    }
    
    // 读线程
    for _ in 0..3 {
        let data = Arc::clone(&data);
        let handle = thread::spawn(move || {
            let r = data.read().unwrap();
            println!("Read: {:?}", *r);
        });
        handles.push(handle);
    }
    
    for handle in handles {
        handle.join().unwrap();
    }
}
```

## Cow<T> - 写时克隆

```rust
use std::borrow::Cow;

fn main() {
    let input = "Hello, World!";
    
    let borrowed: Cow<str> = Cow::Borrowed(input);
    let owned: Cow<str> = Cow::Owned(String::from(input));
    
    // 需要修改时才克隆
    fn process(input: Cow<str>) -> Cow<str> {
        if input.contains("World") {
            let mut s = input.into_owned();
            s.push_str("!");
            Cow::Owned(s)
        } else {
            input
        }
    }
    
    let result = process(borrowed);
    println!("{}", result);
}
```

## 智能指针对比

| 类型 | 所有权 | 线程安全 | 可变性 | 使用场景 |
|------|--------|---------|--------|---------|
| `Box<T>` | 独占 | 是 | 可变 | 堆分配、递归类型 |
| `Rc<T>` | 共享 | 否 | 不可变 | 单线程共享 |
| `Arc<T>` | 共享 | 是 | 不可变 | 多线程共享 |
| `RefCell<T>` | 独占 | 否 | 内部可变 | 单线程内部可变性 |
| `Mutex<T>` | 独占 | 是 | 内部可变 | 多线程可变数据 |
| `RwLock<T>` | 独占 | 是 | 内部可变 | 读多写少场景 |

## Deref 和 Drop Trait

### Deref Trait

```rust
use std::ops::Deref;

struct MyBox<T>(T);

impl<T> MyBox<T> {
    fn new(x: T) -> MyBox<T> {
        MyBox(x)
    }
}

impl<T> Deref for MyBox<T> {
    type Target = T;
    
    fn deref(&self) -> &Self::Target {
        &self.0
    }
}

fn main() {
    let x = 5;
    let y = MyBox::new(x);
    
    // 自动解引用
    assert_eq!(5, *y);
}
```

### Drop Trait

```rust
struct CustomPointer {
    data: String,
}

impl Drop for CustomPointer {
    fn drop(&mut self) {
        println!("Dropping CustomPointer with data: {}", self.data);
    }
}

fn main() {
    let ptr = CustomPointer {
        data: String::from("Hello"),
    };
    // ptr 离开作用域时自动调用 drop
}
```

## 实用示例

### 双向链表

```rust
use std::cell::RefCell;
use std::rc::{Rc, Weak};

#[derive(Debug)]
struct Node {
    value: i32,
    next: RefCell<Option<Rc<Node>>>,
    prev: RefCell<Option<Weak<Node>>>,
}

impl Node {
    fn new(value: i32) -> Rc<Node> {
        Rc::new(Node {
            value,
            next: RefCell::new(None),
            prev: RefCell::new(None),
        })
    }
}

fn main() {
    let a = Node::new(1);
    let b = Node::new(2);
    let c = Node::new(3);
    
    // 连接节点
    *a.next.borrow_mut() = Some(Rc::clone(&b));
    *b.prev.borrow_mut() = Some(Rc::downgrade(&a));
    *b.next.borrow_mut() = Some(Rc::clone(&c));
    *c.prev.borrow_mut() = Some(Rc::downgrade(&b));
}
```

### 缓存系统

```rust
use std::cell::RefCell;
use std::collections::HashMap;
use std::rc::Rc;

struct Cache<K, V> {
    data: RefCell<HashMap<K, V>>,
}

impl<K: std::hash::Hash + Eq + Clone, V: Clone> Cache<K, V> {
    fn new() -> Self {
        Cache {
            data: RefCell::new(HashMap::new()),
        }
    }
    
    fn get(&self, key: &K) -> Option<V> {
        self.data.borrow().get(key).cloned()
    }
    
    fn set(&self, key: K, value: V) {
        self.data.borrow_mut().insert(key, value);
    }
}

fn main() {
    let cache = Cache::new();
    cache.set("key1", "value1");
    println!("{:?}", cache.get(&"key1"));
}
```

## 最佳实践

### 1. 选择正确的智能指针

```rust
// 单线程共享 → Rc
// 多线程共享 → Arc
// 需要内部可变性 → RefCell 或 Mutex
```

### 2. 避免循环引用

```rust
// 使用 Weak 打破循环引用
use std::rc::{Rc, Weak};

struct Node {
    children: Vec<Rc<Node>>,
    parent: Weak<Node>,  // 使用 Weak 避免循环
}
```

### 3. 最小化锁的作用域

```rust
use std::sync::Mutex;

fn good(mutex: &Mutex<Vec<i32>>) {
    let data = mutex.lock().unwrap().clone();
    drop(data);  // 显式释放锁
    // 处理数据...
}
```
