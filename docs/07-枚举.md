# Rust 教程 - 枚举和模式匹配详解

## 本章概述

枚举是定义自定义数据类型的强大工具。本章详细介绍枚举的定义、Option 类型、match 表达式和各种模式匹配技巧。

## 学习目标

- 掌握枚举的定义和使用
- 理解 Option 和 Result 类型
- 掌握 match 表达式
- 学会使用 if let 和 while let
- 理解模式匹配的高级用法

## 1. 枚举定义

### 1.1 基本枚举

```rust
#[derive(Debug)]
enum Direction {
    Up,
    Down,
    Left,
    Right,
}

let up = Direction::Up;
println!("方向: {:?}", up);
```

### 1.2 枚举值关联数据

```rust
#[derive(Debug)]
enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
    ChangeColor(i32, i32, i32),
}
```

## 2. Option 类型

Rust 没有 null，使用 Option<T> 表示可能存在的值：

```rust
let some_number: Option<i32> = Some(5);
let absent_number: Option<i32> = None;
```

## 3. Result 类型

用于错误处理：

```rust
fn divide(a: i32, b: i32) -> Result<i32, String> {
    if b == 0 {
        Err(String::from("除数不能为零"))
    } else {
        Ok(a / b)
    }
}
```

## 4. match 表达式

```rust
match number {
    1 => println!("一"),
    2 | 3 | 5 => println!("小质数"),
    13..=19 => println!("青少年数字"),
    _ => println!("其他"),
}
```

## 5. if let

简化单一模式的匹配：

```rust
if let Some(7) = some_value {
    println!("匹配到 7");
}
```

## 6. while let

循环匹配直到不符合条件：

```rust
while let Some(i) = optional {
    println!("i = {}", i);
    optional = Some(i + 1);
}
```
