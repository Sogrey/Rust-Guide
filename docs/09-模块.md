# Rust 教程 - 包和模块详解

## 本章概述

模块系统是 Rust 组织代码的核心机制。本章详细介绍包、模块、路径和可见性控制。

## 学习目标

- 理解模块系统架构
- 掌握模块定义
- 理解路径和 use 关键字
- 掌握可见性控制
- 学会组织大型项目

## 1. 模块系统概述

- **Package**：Cargo 项目
- **Crate**：模块树
- **Module**：代码组织单元
- **Path**：命名项的方式

## 2. 定义模块

```rust
mod front_of_house {
    pub mod hosting {
        pub fn add_to_waitlist() {}
    }
}
```

## 3. 路径

### 3.1 绝对路径

```rust
crate::front_of_house::hosting::add_to_waitlist();
```

### 3.2 相对路径

```rust
front_of_house::hosting::add_to_waitlist();
```

## 4. use 关键字

引入路径：

```rust
use front_of_house::hosting;

hosting::add_to_waitlist();
```

## 5. pub 关键字

控制可见性：

```rust
pub fn public_function() {}

fn private_function() {}
```

## 6. 文件组织

- `src/lib.rs`：库根
- `src/main.rs`：二进制根
- `src/module.rs`：模块文件
- `src/module/mod.rs`：模块文件（另一种方式）
