# Rust 教程 - 泛型详解

## 本章概述

泛型是编写灵活、可重用代码的核心特性。本章详细介绍泛型函数、结构体、枚举和方法。

## 学习目标

- 掌握泛型函数
- 理解泛型结构体和枚举
- 掌握 Trait Bound
- 理解关联类型
- 了解 Const 泛型

## 1. 泛型函数

```rust
fn largest<T: PartialOrd + Copy>(list: &[T]) -> T {
    let mut largest = list[0];
    for &item in list {
        if item > largest {
            largest = item;
        }
    }
    largest
}
```

## 2. 泛型结构体

```rust
struct Point<T> {
    x: T,
    y: T,
}

let integer_point = Point { x: 5, y: 10 };
let float_point = Point { x: 1.0, y: 4.0 };
```

## 3. 泛型枚举

```rust
enum Container<T> {
    Single(T),
    Pair(T, T),
}
```

## 4. Trait Bound

### 4.1 基本语法

```rust
fn print<T: std::fmt::Display>(item: T) {
    println!("{}", item);
}
```

### 4.2 where 子句

```rust
fn some_function<T, U>(t: T, u: U)
where
    T: Display + Clone,
    U: Clone + Debug,
{
    // ...
}
```

## 5. 关联类型

```rust
trait Iterator {
    type Item;
    fn next(&mut self) -> Option<Self::Item>;
}
```

## 6. Const 泛型

```rust
struct Array<T, const N: usize> {
    data: [T; N],
}
```
