# Rust 教程 - 集合详解

## 本章概述

集合是存储多个数据的数据结构。本章详细介绍 Rust 标准库中的常用集合类型及其操作。

## 学习目标

- 掌握 Vector 的使用
- 理解 String 类型
- 掌握 HashMap 操作
- 了解 HashSet、VecDeque 等其他集合
- 掌握迭代器方法

## 1. Vector (Vec<T>)

动态数组，可增长：

```rust
let mut v = Vec::new();
v.push(1);
v.push(2);

// 或使用宏
let v = vec![1, 2, 3, 4, 5];
```

## 2. String

UTF-8 编码的可增长字符串：

```rust
let mut s = String::from("hello");
s.push_str(" world");
s.push('!');
```

## 3. HashMap

键值对存储：

```rust
use std::collections::HashMap;

let mut scores = HashMap::new();
scores.insert(String::from("Blue"), 10);
scores.insert(String::from("Yellow"), 50);
```

## 4. HashSet

唯一值集合：

```rust
use std::collections::HashSet;

let mut set = HashSet::new();
set.insert(1);
set.insert(2);
```

## 5. 迭代器方法

```rust
let v = vec![1, 2, 3, 4, 5];

// map
let doubled: Vec<i32> = v.iter().map(|x| x * 2).collect();

// filter
let evens: Vec<&i32> = v.iter().filter(|x| *x % 2 == 0).collect();

// fold
let sum = v.iter().fold(0, |acc, x| acc + x);
```
