# Rust 教程 - 错误处理详解

## 本章概述

Rust 将错误分为可恢复和不可恢复两类。本章详细介绍 Rust 的错误处理机制。

## 学习目标

- 理解 panic! 和不可恢复错误
- 掌握 Result<T, E> 类型
- 学会使用 ? 操作符
- 掌握自定义错误类型
- 理解错误处理最佳实践

## 1. panic! - 不可恢复错误

程序立即终止：

```rust
panic!("程序崩溃！");
```

## 2. Result<T, E>

可恢复错误：

```rust
fn divide(a: i32, b: i32) -> Result<i32, String> {
    if b == 0 {
        Err(String::from("除数不能为零"))
    } else {
        Ok(a / b)
    }
}
```

## 3. match 处理 Result

```rust
match divide(10, 2) {
    Ok(result) => println!("结果: {}", result),
    Err(e) => println!("错误: {}", e),
}
```

## 4. unwrap 和 expect

简化处理：

```rust
let result = divide(10, 2).expect("除法失败");
```

## 5. ? 操作符

错误传播：

```rust
fn read_file() -> Result<String, io::Error> {
    let mut s = String::new();
    File::open("file.txt")?.read_to_string(&mut s)?;
    Ok(s)
}
```

## 6. 自定义错误类型

```rust
#[derive(Debug)]
struct MyError {
    message: String,
}

impl std::fmt::Display for MyError {
    fn fmt(&self, f: &mut std::fmt::Formatter) -> std::fmt::Result {
        write!(f, "{}", self.message)
    }
}

impl std::error::Error for MyError {}
```
