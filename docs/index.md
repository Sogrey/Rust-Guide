# Rust 学习指南 - 从入门到精通

欢迎来到 Rust 学习指南！本教程系列将带你从零开始学习 Rust 编程语言，涵盖所有核心概念和实战技巧。

## 项目简介

本项目包含 **27** 个详细的学习章节，每个章节都有配套的源码示例和详细教程文档。所有代码都可以直接运行，帮助你边学边练。

## 快速开始

### 环境准备

1. **安装 Rust**：
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

2. **验证安装**：
```bash
rustc --version
cargo --version
```

### 运行示例

```bash
# 克隆仓库
git clone git@github.com:Sogrey/Rust-Guide.git
cd Rust-Guide

# 运行所有示例
cargo run

# 单独编译检查
cargo check
```

### 学习建议

1. **按顺序学习**：章节之间有依赖关系，建议按顺序学习
2. **动手实践**：每个示例都可以修改和运行，多动手练习
3. **阅读源码**：`src/` 目录下有完整的示例代码
4. **做练习题**：每个章节末尾都有练习题，巩固所学知识

## 核心概念速查

### 所有权三原则

1. 每个值都有一个所有者
2. 同一时刻只能有一个所有者
3. 当所有者离开作用域，值将被丢弃

### 引用规则

1. 可以有多个不可变引用，或者一个可变引用
2. 引用必须始终有效

### 常用类型

| 类型 | 说明 | 示例 |
|------|------|------|
| `i32` | 默认整数类型 | `let x = 42;` |
| `f64` | 默认浮点类型 | `let x = 3.14;` |
| `bool` | 布尔类型 | `let b = true;` |
| `char` | Unicode字符 | `let c = '🦀';` |
| `String` | 堆分配字符串 | `let s = String::from("hello");` |
| `&str` | 字符串切片 | `let s = "hello";` |
| `Vec<T>` | 动态数组 | `let v = vec![1, 2, 3];` |
| `Option<T>` | 可能为空的值 | `Some(5)` 或 `None` |
| `Result<T, E>` | 可能出错的操作 | `Ok(value)` 或 `Err(error)` |

## 常见问题

### Q1: Rust 难学吗？

**答**：Rust 学习曲线较陡峭，主要原因：
- 所有权和借用系统是独特概念
- 编译器很严格，但错误信息友好
- 需要时间适应，但值得投入

建议：
- 不要急躁，理解每个概念
- 多写代码，让编译器成为老师
- 参与社区，寻求帮助

### Q2: 学习 Rust 需要什么基础？

**答**：
- 最好有其他编程语言经验
- 了解基本的编程概念（变量、函数、循环等）
- 有助于但不必须了解：C/C++、函数式编程

### Q3: 学完本教程能做什么？

**答**：
- 编写命令行工具
- Web 后端服务
- 系统编程
- 嵌入式开发
- WebAssembly
- 游戏开发
