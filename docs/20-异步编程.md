# Rust 异步编程

异步编程允许程序在等待 I/O 操作时执行其他任务，提高并发性能。Rust 使用 async/await 语法和 Future trait。

## Future Trait

### Future 基本概念

```rust
use std::future::Future;
use std::pin::Pin;
use std::task::{Context, Poll};

// Future trait 定义
pub trait Future {
    type Output;
    
    fn poll(self: Pin<&mut Self>, cx: &mut Context<'_>) -> Poll<Self::Output>;
}

// Poll 枚举
pub enum Poll<T> {
    Ready(T),
    Pending,
}
```

### 简单 Future

```rust
use std::future::Future;
use std::pin::Pin;
use std::task::{Context, Poll};

struct ReadFileFuture {
    completed: bool,
}

impl Future for ReadFileFuture {
    type Output = String;
    
    fn poll(mut self: Pin<&mut Self>, _cx: &mut Context<'_>) -> Poll<Self::Output> {
        if self.completed {
            Poll::Ready(String::from("File content"))
        } else {
            self.completed = true;
            Poll::Pending
        }
    }
}
```

## async/await 语法

### async 函数

```rust
// async fn 返回 Future
async fn say_hello() {
    println!("Hello, async!");
}

async fn add(a: i32, b: i32) -> i32 {
    a + b
}

fn main() {
    // 需要运行时执行 async 函数
    // 以下代码需要 tokio 或 async-std 运行时
    /*
    #[tokio::main]
    async fn main() {
        say_hello().await;
        let result = add(1, 2).await;
        println!("Result: {}", result);
    }
    */
}
```

### await 表达式

```rust
async fn fetch_data() -> String {
    // 模拟异步操作
    String::from("data")
}

async fn process() {
    // 等待异步操作完成
    let data = fetch_data().await;
    println!("Got: {}", data);
    
    // 链式调用
    println!("Processed: {}", fetch_data().await.to_uppercase());
}
```

## 异步 I/O

### 异步文件操作

```rust
use tokio::fs;

#[tokio::main]
async fn main() -> std::io::Result<()> {
    // 读取文件
    let content = fs::read_to_string("example.txt").await?;
    println!("{}", content);
    
    // 写入文件
    fs::write("output.txt", "Hello, async!").await?;

```

### 异步网络

```rust
use tokio::net::{TcpListener, TcpStream};
use tokio::io::{AsyncReadExt, AsyncWriteExt};

#[tokio::main]
async fn main() -> std::io::Result<()> {
    // TCP 服务器
    let listener = TcpListener::bind("127.0.0.1:8080").await?;
    
    loop {
        let (mut socket, addr) = listener.accept().await?;
        
        tokio::spawn(async move {
            let mut buf = [0; 1024];
            
            loop {
                let n = match socket.read(&mut buf).await {
                    Ok(n) if n == 0 => return,
                    Ok(n) => n,
                    Err(e) => {
                        eprintln!("Error reading: {}", e);
                        return;
                    }
                };
                
                if socket.write_all(&buf[0..n]).await.is_err() {
                    return;
                }
            }
        });
    }
}
```

## 并发任务

### tokio::spawn

```rust
use tokio;

#[tokio::main]
async fn main() {
    let task1 = tokio::spawn(async {
        println!("Task 1");
        1
    });
    
    let task2 = tokio::spawn(async {
        println!("Task 2");
        2
    });
    
    let (result1, result2) = tokio::join!(task1, task2);
    println!("Results: {}, {}", result1.unwrap(), result2.unwrap());
}
```

### tokio::join!

```rust
use tokio;

async fn task1() -> i32 {
    1
}

async fn task2() -> i32 {
    2
}

#[tokio::main]
async fn main() {
    let (r1, r2) = tokio::join!(task1(), task2());
    println!("{} {}", r1, r2);
}
```

### tokio::select!

```rust
use tokio::{self, time::{sleep, Duration}};

#[tokio::main]
async fn main() {
    let task1 = async {
        sleep(Duration::from_millis(100)).await;
        println!("Task 1 done");
        1
    };
    
    let task2 = async {
        sleep(Duration::from_millis(200)).await;
        println!("Task 2 done");
        2
    };
    
    tokio::select! {
        result = task1 => println!("First: {}", result),
        result = task2 => println!("Second: {}", result),
    }
}
```

## 异步通道

### mpsc 通道

```rust
use tokio::sync::mpsc;

#[tokio::main]
async fn main() {
    let (tx, mut rx) = mpsc::channel(32);
    
    tokio::spawn(async move {
        for i in 0..10 {
            tx.send(i).await.unwrap();
        }
    });
    
    while let Some(msg) = rx.recv().await {
        println!("Received: {}", msg);
    }
}
```

### oneshot 通道

```rust
use tokio::sync::oneshot;

#[tokio::main]
async fn main() {
    let (tx, rx) = oneshot::channel();
    
    tokio::spawn(async move {
        tx.send("Hello").unwrap();
    });
    
    let msg = rx.await.unwrap();
    println!("{}", msg);
}
```

### broadcast 通道

```rust
use tokio::sync::broadcast;

#[tokio::main]
async fn main() {
    let (tx, _) = broadcast::channel(16);
    
    let mut rx1 = tx.subscribe();
    let mut rx2 = tx.subscribe();
    
    tx.send("Hello").unwrap();
    
    println!("Rx1: {}", rx1.recv().await.unwrap());
    println!("Rx2: {}", rx2.recv().await.unwrap());
}
```

## 异步同步原语

### Mutex

```rust
use tokio::sync::Mutex;
use std::sync::Arc;

#[tokio::main]
async fn main() {
    let data = Arc::new(Mutex::new(0));
    
    let mut handles = vec![];
    
    for _ in 0..10 {
        let data = Arc::clone(&data);
        let handle = tokio::spawn(async move {
            let mut lock = data.lock().await;
            *lock += 1;
        });
        handles.push(handle);
    }
    
    for handle in handles {
        handle.await.unwrap();
    }
    
    println!("Result: {}", *data.lock().await);
}
```

### RwLock

```rust
use tokio::sync::RwLock;
use std::sync::Arc;

#[tokio::main]
async fn main() {
    let data = Arc::new(RwLock::new(vec![1, 2, 3]));
    
    // 读
    {
        let r = data.read().await;
        println!("{:?}", *r);
    }
    
    // 写
    {
        let mut w = data.write().await;
        w.push(4);
    }
}
```

## 错误处理

### 异步错误

```rust
use tokio;

async fn might_fail() -> Result<i32, String> {
    Ok(42)
}

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let result = might_fail().await?;
    println!("{}", result);
    Ok(())
}
```

### 超时处理

```rust
use tokio::time::{timeout, Duration};

async fn slow_operation() -> String {
    tokio::time::sleep(Duration::from_secs(2)).await;
    String::from("Done")
}

#[tokio::main]
async fn main() {
    match timeout(Duration::from_secs(1), slow_operation()).await {
        Ok(result) => println!("Success: {}", result),
        Err(_) => println!("Timeout!"),
    }
}
```

## Stream

### Stream trait

```rust
use futures::stream::{self, StreamExt};

#[tokio::main]
async fn main() {
    let mut stream = stream::iter(1..=5);
    
    while let Some(item) = stream.next().await {

```

### Stream 操作

```rust
use futures::stream::{self, StreamExt};

#[tokio::main]
async fn main() {
    let result: Vec<i32> = stream::iter(1..=10)
        .filter(|x| async move { x % 2 == 0 })
        .map(|x| x * 2)
        .collect()
        .await;
    
    println!("{:?}", result);
}
```

## 最佳实践

### 1. 避免阻塞异步运行时

```rust
// ❌ 阻塞运行时
async fn bad() {
    std::thread::sleep(std::time::Duration::from_secs(1));
}

// ✅ 使用异步 sleep
async fn good() {
    tokio::time::sleep(std::time::Duration::from_secs(1)).await;
}
```

### 2. 使用 spawn_blocking 处理 CPU 密集任务

```rust
#[tokio::main]
async fn main() {
    let result = tokio::task::spawn_blocking(|| {
        // CPU 密集计算
        (1..=1_000_000).sum::<u64>()
    }).await.unwrap();
    
    println!("{}", result);
}
```

### 3. 合理使用并发

```rust
use tokio;

#[tokio::main]
async fn main() {
    // 控制并发数量
    let sem = Arc::new(tokio::sync::Semaphore::new(3));
    
    let mut handles = vec![];
    
    for i in 0..10 {
        let sem = Arc::clone(&sem);
        let handle = tokio::spawn(async move {
            let _permit = sem.acquire().await.unwrap();
            println!("Task {}", i);
        });
        handles.push(handle);
    }
    
    for handle in handles {
        handle.await.unwrap();
    }
}
```

## 异步 vs 线程

| 特性 | 异步 | 线程 |
|------|------|------|
| 内存开销 | 低 | 高 |
| 上下文切换 | 快 | 慢 |
| 编程复杂度 | 高 | 低 |
| CPU 密集 | 需要特殊处理 | 适合 |
| I/O 密集 | 适合 | 可用 |
