# Rust 教程 - 函数详解

## 本章概述

函数是 Rust 代码的组织单元。本章详细介绍函数的定义、参数、返回值、闭包等概念，以及 Rust 特有的表达式特性。

## 学习目标

- 掌握函数定义和调用
- 理解参数和返回值
- 区分语句和表达式
- 掌握闭包的使用
- 理解函数指针
- 学会使用方法

## 1. 函数定义

使用 `fn` 关键字定义函数：

```rust
fn basic_function() {
    println!("这是一个基础函数");
}

fn main() {
    basic_function();
}
```

**命名规则**：
- 使用 snake_case 命名风格
- 函数名应具有描述性
- 避免过长的名称

## 2. 函数参数

### 2.1 基本参数

参数必须声明类型：

```rust
fn print_labeled_measurement(value: i32, unit_label: char) {
    println!("测量值: {}{}", value, unit_label);
}
```

### 2.2 参数模式

支持模式匹配：

```rust
fn print_coordinates(&(x, y): &(i32, i32)) {
    println!("当前位置: ({}, {})", x, y);
}

let point = (3, 5);
print_coordinates(&point);
```

## 3. 返回值

### 3.1 显式返回

使用 `return` 关键字：

```rust
fn add_five(x: i32) -> i32 {
    return x + 5;
}
```

### 3.2 隐式返回

最后一个表达式的值自动返回：

```rust
fn another_add(x: i32, y: i32) -> i32 {
    x + y  // 注意：没有分号！这是一个表达式
}
```

**重要**：表达式没有分号，加上分号就变成了语句。

### 3.3 无返回值

返回单元类型 `()`：

```rust
fn no_return() {
    println!("没有返回值");
    // 隐式返回 ()
}

fn explicit_no_return() -> () {
    println!("显式声明无返回值");
}
```

## 4. 语句 vs 表达式

这是 Rust 的重要概念。

### 语句（Statement）

执行操作但不返回值：

```rust
let x = 5;  // 语句
```

### 表达式（Expression）

计算并返回值：

```rust
let y = {
    let x = 3;
    x + 1  // 表达式，返回 4
};
```

**代码块作为表达式**：

```rust
let condition = true;
let number = if condition { 5 } else { 6 };  // if 是表达式

let result = loop {
    break 10;  // break 可以返回值
};
```

## 5. 闭包（Closure）

闭包是可以捕获环境的匿名函数。

### 5.1 基本语法

```rust
let add_one = |x: i32| -> i32 { x + 1 };
println!("闭包调用: {}", add_one(5));
```

### 5.2 类型推断

闭包会自动推断类型：

```rust
let add_two = |x| x + 2;  // 自动推断类型
println!("简化闭包: {}", add_two(5));
```

### 5.3 捕获环境

闭包可以捕获外部变量：

```rust
let multiplier = 10;
let multiply = |x| x * multiplier;
println!("捕获环境变量: {}", multiply(5));
```

### 5.4 闭包的三种捕获方式

| 方式 | 关键字 | 类似于 |
|------|--------|--------|
| 不可变借用 | 默认 | &T |
| 可变借用 | `mut` | &mut T |
| 获取所有权 | `move` | T |

```rust
// 不可变借用
let x = 10;
let closure = || println!("{}", x);

// 可变借用
let mut counter = 0;
let mut increment = || {
    counter += 1;
    counter
};

// 获取所有权
let name = String::from("Rust");
let closure = move || println!("{}", name);
```

## 6. 函数指针

函数可以作为值传递。

### 6.1 函数类型

```rust
fn add_five(x: i32) -> i32 {
    x + 5
}

let fn_ptr: fn(i32) -> i32 = add_five;
let result = fn_ptr(10);
println!("函数指针调用: {}", result);
```

### 6.2 函数作为参数

```rust
fn apply_function(value: i32, f: fn(i32) -> i32) {
    println!("应用函数: f({}) = {}", value, f(value));
}

apply_function(5, add_five);
```

## 7. 高阶函数

返回函数的函数：

```rust
fn create_adder(x: i32) -> fn(i32) -> i32 {
    add_five  // 返回函数指针
}
```

返回闭包：

```rust
fn create_multiplier(factor: i32) -> Box<dyn Fn(i32) -> i32> {
    Box::new(move |x| x * factor)
}

let multiplier = create_multiplier(3);
println!("返回闭包的调用: {}", multiplier(5));
```

## 8. 方法

方法定义在 `impl` 块中。

### 8.1 实例方法

```rust
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    fn area(&self) -> u32 {
        self.width * self.height
    }
}

let rect = Rectangle { width: 10, height: 20 };
println!("面积: {}", rect.area());
```

### 8.2 关联函数

没有 `self` 参数，类似静态方法：

```rust
impl Rectangle {
    fn square(size: u32) -> Rectangle {
        Rectangle {
            width: size,
            height: size,
        }
    }
}

let square = Rectangle::square(5);
```

### 8.3 多个 impl 块

可以为同一类型定义多个 `impl` 块：

```rust
impl Rectangle {
    fn width(&self) -> u32 {
        self.width
    }
}

impl Rectangle {
    fn height(&self) -> u32 {
        self.height
    }
}
```

## 9. 递归函数

函数可以调用自身：

```rust
// 阶乘
fn factorial(n: u64) -> u64 {
    if n <= 1 {
        1
    } else {
        n * factorial(n - 1)
    }
}

// 斐波那契
fn fibonacci(n: u64) -> u64 {
    if n <= 1 {
        n
    } else {
        fibonacci(n - 1) + fibonacci(n - 2)
    }
}
```

## 10. 发散函数

发散函数永远不会返回，返回类型是 `!`：

```rust
fn always_panic() -> ! {
    panic!("永远不返回");
}

fn may_panic(x: i32) -> i32 {
    if x == 0 {
        panic!("值为 0！");  // panic! 永远不返回
    }
    x * 2
}
```

**用途**：
- `panic!` 宏
- `std::process::exit`
- 死循环

## 11. 常量函数

使用 `const fn` 定义可以在编译时计算的函数：

```rust
const fn compute_at_compile(x: i32) -> i32 {
    x * x * x
}

const COMPILE_TIME_VALUE: i32 = compute_at_compile(5);
```

**限制**：
- 只能调用常量函数
- 不能使用循环（某些情况下）
- 不能使用堆分配

## 12. 最佳实践

### 函数命名

```rust
// 好 - 动词开头，描述性强
fn calculate_total_price(items: &[Item]) -> f64 { /* ... */ }
fn send_email(to: &str, subject: &str) { /* ... */ }

// 避免
fn price(items: &[Item]) -> f64 { /* ... */ }
fn email(to: &str, subject: &str) { /* ... */ }
```

### 参数数量

```rust
// 好 - 参数少，使用结构体
struct Config {
    host: String,
    port: u16,
    timeout: u64,
}

fn connect(config: Config) { /* ... */ }

// 避免 - 参数过多
fn connect(host: String, port: u16, timeout: u64, /* ... */) { /* ... */ }
```

### 避免副作用

```rust
// 好 - 纯函数，无副作用
fn add(a: i32, b: i32) -> i32 {
    a + b
}

// 避免 - 副用
fn add_and_print(a: i32, b: i32) -> i32 {
    let result = a + b;
    println!("{}", result);  // 副作用
    result
}
```

## 常见问题

### Q1: 函数和闭包有什么区别？

**答**：
- 函数：定义在模块级别，不能直接捕获环境
- 闭包：匿名，可以捕获环境，类型推断更灵活

### Q2: 什么时候使用 `return`？

**答**：
- 提前返回时必须使用
- 最后返回时可以省略（推荐）

### Q3: 方法中的 `self` 有哪些形式？

**答**：
- `&self`：不可变借用
- `&mut self`：可变借用
- `self`：获取所有权
