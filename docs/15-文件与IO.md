# Rust 文件与 IO

文件操作和输入输出是程序与外部世界交互的重要方式。Rust 提供了强大且安全的 IO 操作 API。

## 标准输入输出

### 标准输出

```rust
use std::io::{self, Write};

fn main() -> io::Result<()> {
    // 基本输出
    println!("Hello, World!");
    print!("No newline");
    print!(" - continue\n");
    
    // 格式化输出
    let name = "Alice";
    let age = 30;
    println!("Name: {}, Age: {}", name, age);
    
    // 输出到标准错误
    eprintln!("Error message");
    eprint!("Warning: ");
    eprintln!("Something went wrong");
    
    // 刷新缓冲区
    print!("Processing...");
    io::stdout().flush()?;
    // 执行长时间操作
    println!(" Done");
    
    Ok(())
}
```

### 标准输入

```rust
use std::io;

fn main() -> io::Result<()> {
    // 读取一行
    println!("Enter your name:");
    let mut input = String::new();
    io::stdin().read_line(&mut input)?;
    println!("Hello, {}", input.trim());
    
    // 读取多行
    println!("Enter multiple lines (empty line to finish):");
    loop {
        let mut line = String::new();
        io::stdin().read_line(&mut line)?;
        let line = line.trim();
        if line.is_empty() {
            break;
        }
        println!("You entered: {}", line);
    }
    
    Ok(())
}
```

### 解析输入

```rust
use std::io;

fn main() {
    // 读取并解析整数
    println!("Enter a number:");
    let mut input = String::new();
    io::stdin().read_line(&mut input).expect("Failed to read");
    
    let number: i32 = input.trim().parse().expect("Invalid number");
    println!("You entered: {}", number);
    
    // 安全的解析方式
    println!("Enter another number:");
    let mut input2 = String::new();
    io::stdin().read_line(&mut input2).expect("Failed to read");
    
    match input2.trim().parse::<i32>() {
        Ok(num) => println!("Valid number: {}", num),
        Err(e) => println!("Invalid number: {}", e),
    }
}
```

## 文件操作

### 打开文件

```rust
use std::fs::File;
use std::io::prelude::*;

fn main() -> std::io::Result<()> {
    // 打开文件（只读）
    let file = File::open("example.txt")?;
    
    // 打开文件（写入，不存在则创建）
    let file = File::create("output.txt")?;
    
    // 打开文件（追加模式）
    use std::fs::OpenOptions;
    let file = OpenOptions::new()
        .write(true)
        .append(true)
        .open("output.txt")?;
    
    // 打开文件（读写模式）
    let file = OpenOptions::new()
        .read(true)
        .write(true)
        .create(true)
        .open("data.txt")?;
    
    Ok(())
}
```

#### 带缓冲读取

```rust
use std::fs::File;
use std::io::{BufRead, BufReader};

fn main() -> std::io::Result<()> {
    let file = File::open("example.txt")?;
    let reader = BufReader::new(file);
    
    // 按字节读取
    for byte_result in reader.bytes() {
        let byte = byte_result?;
        println!("Byte: {}", byte);
    }
    
    // 按字符读取
    let file = File::open("example.txt")?;
    let reader = BufReader::new(file);
    for char_result in reader.chars() {
        let ch = char_result?;
        print!("{}", ch);
    }
    
    Ok(())
}
```

#### 分割读取

```rust
use std::fs::File;
use std::io::{BufRead, BufReader};

fn main() -> std::io::Result<()> {
    let file = File::open("example.txt")?;
    let reader = BufReader::new(file);
    
    // 按分隔符分割（默认换行符）
    for segment in reader.split(b'\n') {
        let segment = segment?;
        println!("{:?}", String::from_utf8_lossy(&segment));
    }
    
    // 按 take 读取固定大小
    let file = File::open("example.txt")?;
    let mut reader = BufReader::new(file);
    let mut buffer = [0; 10];
    
    loop {
        let bytes_read = reader.read(&mut buffer)?;
        if bytes_read == 0 {
            break;
        }
        println!("Read {} bytes: {:?}", bytes_read, &buffer[..bytes_read]);
    }
    
    Ok(())
}
```

### 文件元数据

```rust
use std::fs;

fn main() -> std::io::Result<()> {
    let metadata = fs::metadata("example.txt")?;
    
    println!("File size: {} bytes", metadata.len());
    println!("Is file: {}", metadata.is_file());
    println!("Is directory: {}", metadata.is_dir());
    println!("Is symlink: {}", metadata.is_symlink());
    
    // 权限
    let permissions = metadata.permissions();
    println!("Readonly: {}", permissions.readonly());
    
    // 修改时间
    if let Ok(modified) = metadata.modified() {
        println!("Modified: {:?}", modified);
    }
    
    // 访问时间
    if let Ok(accessed) = metadata.accessed() {
        println!("Accessed: {:?}", accessed);
    }
    
    Ok(())
}
```

### 列出目录内容

```rust
use std::fs;

fn main() -> std::io::Result<()> {
    // 读取目录
    let entries = fs::read_dir(".")?;
    
    for entry in entries {
        let entry = entry?;
        let path = entry.path();
        
        // 获取文件名
        if let Some(name) = path.file_name() {
            println!("{:?}", name);
        }
        
        // 检查类型
        let file_type = entry.file_type()?;
        if file_type.is_file() {
            println!("  File: {:?}", path);
        } else if file_type.is_dir() {
            println!("  Dir: {:?}", path);
        }
    }
    
    Ok(())
}
```

## 文件路径操作

### Path 和 PathBuf

```rust
use std::path::{Path, PathBuf};

fn main() {
    // 创建路径
    let path = Path::new("src/main.rs");
    println!("Path: {:?}", path);
    
    // PathBuf - 可变的路径
    let mut path_buf = PathBuf::new();
    path_buf.push("src");
    path_buf.push("main.rs");
    println!("PathBuf: {:?}", path_buf);
    
    // 从字符串创建
    let path = PathBuf::from("src/lib.rs");
    
    // 路径组件
    let path = Path::new("/usr/local/bin/rust");
    
    // 父目录
    println!("Parent: {:?}", path.parent());  // Some("/usr/local/bin")
    
    // 文件名
    println!("File name: {:?}", path.file_name());  // Some("rust")
    
    // 扩展名
    let path = Path::new("main.rs");
    println!("Extension: {:?}", path.extension());  // Some("rs")
    
    // 文件名（不含扩展名）
    println!("File stem: {:?}", path.file_stem());  // Some("main")
    
    // 检查路径
    println!("Exists: {}", path.exists());
    println!("Is file: {}", path.is_file());
    println!("Is dir: {}", path.is_dir());
    
    // 连接路径
    let path1 = Path::new("/usr/local");
    let path2 = path1.join("bin").join("rust");
    println!("Joined: {:?}", path2);
    
    // 绝对路径
    if let Ok(abs_path) = std::fs::canonicalize(".") {
        println!("Absolute path: {:?}", abs_path);
    }
}
```

## 缓冲和性能

### BufReader 和 BufWriter

```rust
use std::fs::File;
use std::io::{BufReader, BufWriter, Read, Write};

fn main() -> std::io::Result<()> {
    // 读取大文件
    let file = File::open("large_file.txt")?;
    let mut reader = BufReader::with_capacity(8192, file);
    
    let mut buffer = [0; 1024];
    loop {
        let bytes_read = reader.read(&mut buffer)?;
        if bytes_read == 0 {
            break;
        }
        // 处理数据
    }
    
    // 写入大文件
    let file = File::create("output.txt")?;
    let mut writer = BufWriter::with_capacity(8192, file);
    
    for i in 0..1000 {
        writeln!(writer, "Line {}", i)?;
    }
    
    writer.flush()?;
    
    Ok(())
}
```

## 错误处理

### 处理 IO 错误

```rust
use std::fs::File;
use std::io::ErrorKind;

fn main() {
    // 使用 match 处理不同错误
    let file = File::open("nonexistent.txt");
    
    match file {
        Ok(file) => {
            println!("File opened: {:?}", file);
        }
        Err(error) => match error.kind() {
            ErrorKind::NotFound => {
                println!("File not found, creating...");
                // 创建文件
            }
            ErrorKind::PermissionDenied => {
                println!("Permission denied");
            }
            _ => {
                println!("Error: {}", error);
            }
        }
    }
    
    // 使用 unwrap_or_else
    let file = File::open("example.txt").unwrap_or_else(|error| {
        if error.kind() == ErrorKind::NotFound {
            File::create("example.txt").unwrap()
        } else {
            panic!("Error opening file: {:?}", error);
        }
    });
}
```

### 自定义 IO 错误

```rust
use std::io;
use std::fmt;

#[derive(Debug)]
enum MyError {
    Io(io::Error),
    Custom(String),
}

impl fmt::Display for MyError {
    fn fmt(&self, f: &mut fmt::Formatter) -> fmt::Result {
        match self {
            MyError::Io(e) => write!(f, "IO error: {}", e),
            MyError::Custom(s) => write!(f, "Custom error: {}", s),
        }
    }
}

impl From<io::Error> for MyError {
    fn from(error: io::Error) -> Self {
        MyError::Io(error)
    }
}

fn read_config() -> Result<String, MyError> {
    use std::fs;
    let content = fs::read_to_string("config.txt")?;
    
    if content.is_empty() {
        return Err(MyError::Custom("Config is empty".to_string()));
    }
    
    Ok(content)
}

fn main() {
    match read_config() {
        Ok(content) => println!("Config: {}", content),
        Err(e) => eprintln!("Error: {}", e),
    }
}
```

## 实用示例

### 配置文件读取

```rust
use std::fs;
use std::collections::HashMap;
use std::io::{self, BufRead};

fn read_config(path: &str) -> io::Result<HashMap<String, String>> {
    let file = fs::File::open(path)?;
    let reader = io::BufReader::new(file);
    let mut config = HashMap::new();
    
    for line in reader.lines() {
        let line = line?;
        let line = line.trim();
        
        // 跳过注释和空行
        if line.is_empty() || line.starts_with('#') {
            continue;
        }
        
        // 解析 key=value
        let parts: Vec<&str> = line.splitn(2, '=').collect();
        if parts.len() == 2 {
            config.insert(parts[0].trim().to_string(), parts[1].trim().to_string());
        }
    }
    
    Ok(config)
}

fn main() -> io::Result<()> {
    let config = read_config("config.txt")?;
    
    for (key, value) in &config {
        println!("{} = {}", key, value);
    }
    
    Ok(())
}
```

### 文件搜索

```rust
use std::fs;
use std::io;

fn search_in_file(path: &str, pattern: &str) -> io::Result<Vec<usize>> {
    let content = fs::read_to_string(path)?;
    let mut line_numbers = Vec::new();
    
    for (i, line) in content.lines().enumerate() {
        if line.contains(pattern) {
            line_numbers.push(i + 1);
        }
    }
    
    Ok(line_numbers)
}

fn main() -> io::Result<()> {
    let matches = search_in_file("example.txt", "Rust")?;
    
    println!("Pattern found on lines: {:?}", matches);
    
    Ok(())
}
```

### 文件复制工具

```rust
use std::fs::{self, File};
use std::io::{self, BufReader, BufWriter, Read, Write};
use std::path::Path;

fn copy_file(source: &Path, dest: &Path) -> io::Result<u64> {
    // 检查源文件
    if !source.exists() {
        return Err(io::Error::new(io::ErrorKind::NotFound, "Source not found"));
    }
    
    // 检查目标目录
    if let Some(parent) = dest.parent() {
        fs::create_dir_all(parent)?;
    }
    
    // 复制文件
    let src_file = File::open(source)?;
    let dest_file = File::create(dest)?;
    
    let mut reader = BufReader::new(src_file);
    let mut writer = BufWriter::new(dest_file);
    
    let bytes = io::copy(&mut reader, &mut writer)?;
    writer.flush()?;
    
    Ok(bytes)
}

fn main() -> io::Result<()> {
    let source = Path::new("source.txt");
    let dest = Path::new("backup/source.txt");
    
    let bytes = copy_file(source, dest)?;
    println!("Copied {} bytes", bytes);
    
    Ok(())
}
```
