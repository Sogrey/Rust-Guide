# Rust 输出到命令行

输出到命令行是最基本的操作之一，Rust 提供了多种方式来格式化和输出文本。

## 基本输出

### print! 和 println!

`print!` 宏用于输出文本，不换行：
`println!` 宏用于输出文本并换行：

```rust
fn main() {
    // 输出不换行
    print!("Hello, ");
    print!("World!");
    
    // 输出并换行
    println!("Hello, World!");
    println!("This is a new line");
}
```

输出：
```
Hello, World!Hello, World!
This is a new line
```

### 基本用法

```rust
fn main() {
    // 空行
    println!();
    
    // 字符串字面量
    println!("Hello, Rust!");
    
    // 包含转移字符
    println!("Line 1\nLine 2");     // \n 换行
    println!("Tab:\tvalue");         // \t 制表符
    println!("Quote: \"hello\"");    // \" 双引号
    println!("Backslash: \\");       // \\ 反斜杠
}
```

## 格式化输出

### 基本占位符

Rust 使用 `{}` 作为占位符：

```rust
fn main() {
    let name = "Alice";
    let age = 30;
    
    // 基本占位符
    println!("Name: {}", name);
    println!("Age: {}", age);
    
    // 多个占位符
    println!("Name: {}, Age: {}", name, age);
    
    // 占位符数量必须匹配
    // println!("{}, {}", name);  // 错误！占位符太多
    // println!("{}", name, age); // 错误！参数太多
}
```

### 位置参数

使用索引指定参数位置：

```rust
fn main() {
    println!("{0} {1} {0}", "a", "b");
    // 输出: a b a
    
    // 可以乱序使用
    println!("{2} {1} {0}", "first", "second", "third");
    // 输出: third second first
}
```

### 命名参数

使用命名参数提高可读性：

```rust
fn main() {
    println!("{name} is {age} years old", 
             name = "Alice", 
             age = 30);
    
    // 混合使用
    println!("{0} is {age} years old", 
             "Bob", 
             age = 25);
}
```

### 格式化特性

Rust 类型可以实现不同的格式化特性：

```rust
fn main() {
    let value = 42;
    
    // Display 特性 - {} - 友好显示
    println!("Display: {}", value);
    
    // Debug 特性 - {:?} - 调试格式
    let arr = [1, 2, 3];
    println!("Debug: {:?}", arr);
    
    // 美化的 Debug - {:#?}
    let person = Person { name: "Alice", age: 30 };
    println!("Pretty Debug:\n{:#?}", person);
    
    // 二进制格式
    println!("Binary: {:b}", value);
    
    // 八进制格式
    println!("Octal: {:o}", value);
    
    // 十六进制格式（小写）
    println!("Hex (lower): {:x}", value);
    
    // 十六进制格式（大写）
    println!("Hex (upper): {:X}", value);
    
    // 指数格式
    let float_val = 12345.6789;
    println!("Exponential: {:e}", float_val);
}

#[derive(Debug)]
struct Person {
    name: &'static str,
    age: u32,
}
```

## 数值格式化

### 宽度和对齐

```rust
fn main() {
    let value = 42;
    
    // 最小宽度
    println!("{:5}", value);      // "   42"
    
    // 左对齐
    println!("{:<5}", value);     // "42   "
    
    // 右对齐（默认）
    println!("{:>5}", value);     // "   42"
    
    // 居中对齐
    println!("{:^5}", value);     // " 42  "
    
    // 带符号
    println!("{:+5}", value);     // "  +42"
    
    // 带填充字符
    println!("{:0>5}", value);    // "00042"
    println!("{:*^5}", value);    // "*42**"
}
```

### 精度控制

```rust
fn main() {
    let pi = 3.14159265359;
    
    // 小数位数
    println!("{:.2}", pi);        // "3.14"
    println!("{:.5}", pi);        // "3.14159"
    
    // 宽度 + 精度
    println!("{:8.2}", pi);       // "    3.14"
    println!("{:<8.2}", pi);      // "3.14    "
    
    // 字符串精度（截断）
    let s = "Hello, World!";
    println!("{:.5}", s);         // "Hello"
}
```

### 数字格式

```rust
fn main() {
    let value = 42;
    
    // 二进制
    println!("Binary: {:b}", value);           // "101010"
    println!("Binary: {:08b}", value);         // "00101010"
    
    // 八进制
    println!("Octal: {:o}", value);            // "52"
    println!("Octal: {:04o}", value);          // "0052"
    
    // 十六进制
    println!("Hex: {:x}", value);              // "2a"
    println!("Hex: {:X}", value);              // "2A"
    println!("Hex: {:04x}", value);            // "002a"
    println!("Hex: {:#x}", value);             // "0x2a"（带前缀）
    
    // 数字分隔符（Rust 1.67+）
    let big_num = 1_000_000;
    println!("Big number: {}", big_num);       // "1000000"
}
```

## format! 宏

`format!` 宏用于创建格式化字符串，而不输出：

```rust
fn main() {
    let name = "Alice";
    let age = 30;
    
    // 创建字符串
    let message = format!("Name: {}, Age: {}", name, age);
    println!("{}", message);
    
    // 使用 format! 构建复杂字符串
    let s1 = format!("{:-^20}", "Title");
    let s2 = format!("Name: {:10}", name);
    let s3 = format!("Age: {:10}", age);
    
    println!("{}", s1);
    println!("{}", s2);
    println!("{}", s3);
}
```

## eprint! 和 eprintln!

输出到标准错误（stderr）：

```rust
fn main() {
    // 标准输出（stdout）
    println!("This is normal output");
    
    // 标准错误（stderr）
    eprintln!("This is an error message");
    
    // 常用于日志和错误信息
    eprintln!("[ERROR] File not found: {}", "data.txt");
    eprintln!("[WARNING] Low memory");
}
```

## 高级格式化

### 自定义格式化

通过实现 `Display` 和 `Debug` trait 来自定义格式化：

```rust
use std::fmt;

struct Point {
    x: i32,
    y: i32,
}

// 实现 Display trait（用户友好）
impl fmt::Display for Point {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        write!(f, "({}, {})", self.x, self.y)
    }
}

// 实现 Debug trait（调试用）
impl fmt::Debug for Point {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        write!(f, "Point {{ x: {}, y: {} }}", self.x, self.y)
    }
}

fn main() {
    let p = Point { x: 10, y: 20 };
    
    println!("Display: {}", p);     // Display: (10, 20)
    println!("Debug: {:?}", p);     // Debug: Point { x: 10, y: 20 }
}
```

### 二进制、八进制、十六进制的自定义

```rust
use std::fmt;

struct Color {
    r: u8,
    g: u8,
    b: u8,
}

impl fmt::Display for Color {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        write!(f, "RGB({}, {}, {})", self.r, self.g, self.b)
    }
}

impl fmt::LowerHex for Color {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        write!(f, "#{:02x}{:02x}{:02x}", self.r, self.g, self.b)
    }
}

impl fmt::UpperHex for Color {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        write!(f, "#{:02X}{:02X}{:02X}", self.r, self.g, self.b)
    }
}

fn main() {
    let color = Color { r: 255, g: 128, b: 0 };
    
    println!("Display: {}", color);         // RGB(255, 128, 0)
    println!("Hex: {:x}", color);           // #ff8000
    println!("Hex: {:X}", color);           // #FF8000
}
```

### 条件格式化

```rust
fn main() {
    let value = Some(42);
    
    // 使用 if 表达式
    println!("{}", if value.is_some() { "有值" } else { "无值" });
    
    // 使用 match 表达式
    println!("{}", match value {
        Some(v) => format!("值: {}", v),
        None => "无值".to_string(),
    });
}
```

## 转义和原始字符串

### 转义字符

```rust
fn main() {
    println!("Line 1\nLine 2");         // 换行
    println!("Tab:\tvalue");             // 制表符
    println!("Quote: \"hello\"");        // 双引号
    println!("Single quote: \'");        // 单引号
    println!("Backslash: \\");           // 反斜杠
    println!("Unicode: \u{1F600}");      // Unicode 字符 😀
}
```

### 原始字符串

```rust
fn main() {
    // 原始字符串（不转义）
    println!(r"Hello\nWorld");           // 输出: Hello\nWorld
    println!(r#"Quote: "hello""#);       // 输出: Quote: "hello"
    println!(r##"Hash: #symbol"##);      // 输出: Hash: #symbol
    
    // 原始字符串用于正则表达式、文件路径等
    let regex = r"\d+";
    let path = r"C:\Users\name\file.txt";
    println!("Regex: {}", regex);
    println!("Path: {}", path);
}
```

## 性能考虑

### 减少不必要的格式化

```rust
fn main() {
    // 好：直接输出
    println!("Hello, World!");
    
    // 不必要的格式化
    println!("{}", "Hello, World!");  // 可以但不必要
    
    // 必要的格式化
    let name = "Alice";
    println!("Hello, {}!", name);
}
```

### 批量输出

```rust
fn main() {
    // 好：使用单个 println!
    println!("Line 1\nLine 2\nLine 3");
    
    // 较慢：多个 println!
    println!("Line 1");
    println!("Line 2");
    println!("Line 3");
}
```

### 使用缓冲

对于大量输出，使用缓冲可以提高性能：

```rust
use std::io::{self, BufWriter, Write};

fn main() -> io::Result<()> {
    let stdout = io::stdout();
    let mut buffer = BufWriter::new(stdout.lock());
    
    for i in 0..10000 {
        writeln!(buffer, "Line {}", i)?;
    }
    
    Ok(())
}
```

## 实用示例

### 表格输出

```rust
fn main() {
    // 表头
    println!("{:<10} {:>10} {:>10}", "Name", "Score", "Grade");
    println!("{:-<10} {:->10} {:->10}", "", "", "");
    
    // 数据行
    println!("{:<10} {:>10} {:>10}", "Alice", 95, "A");
    println!("{:<10} {:>10} {:>10}", "Bob", 87, "B");
    println!("{:<10} {:>10} {:>10}", "Charlie", 92, "A");
}
```

输出：
```
Name            Score      Grade
---------- ---------- ----------
Alice               95          A
Bob                 87          B
Charlie             92          A
```

### 进度条

```rust
fn main() {
    for i in 0..=100 {
        print!("\rProgress: [");
        let completed = i / 5;
        for j in 0..20 {
            if j < completed {
                print!("=");
            } else if j == completed {
                print!(">");
            } else {
                print!(" ");
            }
        }
        print!("] {}%", i);
        
        // 模拟工作
        std::thread::sleep(std::time::Duration::from_millis(50));
    }
    println!();
}
```

### 日志格式

use chrono::Local;

fn log(level: &str, message: &str) {

| `{:?}` | Debug 特性 | `println!("{:?}", vec)` |
| `{:#?}` | 美化 Debug | `println!("{:#?}", struct)` |
| `{:b}` | 二进制 | `println!("{:b}", 42)` → "101010" |
| `{:o}` | 八进制 | `println!("{:o}", 42)` → "52" |
| `{:x}` | 十六进制（小写） | `println!("{:x}", 42)` → "2a" |
| `{:X}` | 十六进制（大写） | `println!("{:X}", 42)` → "2A" |
| `{:e}` | 指数（小写） | `println!("{:e}", 1000.0)` → "1e3" |
| `{:E}` | 指数（大写） | `println!("{:E}", 1000.0)` → "1E3" |

## 常见问题

### 1. 类型没有实现 Display

**错误**：`the trait 'Display' is not implemented for 'XXX'`

**解决方案**：
- 使用 `{:?}` (Debug) 代替
- 为类型实现 Display trait

```rust
fn main() {
    let v = vec![1, 2, 3];
    // println!("{}", v);  // 错误！
    println!("{:?}", v);   // 正确
}
```

### 2. 占位符数量不匹配

**错误**：`argument never used` 或 `expected 2 arguments`

**解决方案**：确保占位符数量和参数数量匹配

```rust
fn main() {
    // 错误
    // println!("{}, {}", 1);      // 参数太少
    // println!("{}", 1, 2);       // 参数太多
    
    // 正确
    println!("{}, {}", 1, 2);      // 正确
    println!("{0}, {0}", 1);       // 重复使用参数
}
```

### 3. 输出缓冲

**问题**：输出没有立即显示

**解决方案**：
- `println!` 会自动刷新缓冲区
- `print!` 需要手动刷新或使用 `flush()`

```rust
use std::io::{self, Write};

fn main() -> io::Result<()> {
    print!("Processing...");
    io::stdout().flush()?;  // 立即显示
    // 执行长时间操作
    println!(" Done");
    Ok(())
}
```
