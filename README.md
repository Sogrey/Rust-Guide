# Rust 学习指南 - 从入门到精通

Rust 语言完整学习示例，从入门到精通。

## 项目简介

本项目包含 **27 个详细的学习章节**，涵盖了 Rust 语言从基础到高级的所有核心概念和特性。每个章节都包含丰富的代码示例和详细的中文注释，配套完整的教程文档，适合初学者系统学习 Rust。

> 我想要学习 Rust，帮我创建出各种 .rs 文件，命名带上数字序号，详细说明 Rust 中的各种语法和功能，包括：环境搭建、Cargo、基础语法、运算符、数据类型、注释、函数、条件语句、循环、迭代器、闭包、所有权、Slice（切片）类型、结构体、枚举类、组织管理、错误处理、泛型与特性、生命周期、文件与 IO、集合与字符串、面向对象、并发编程、宏、智能指针、异步编程等。

## 学习内容

### 第零阶段：环境准备（5章）

#### 00. 环境搭建
- Windows/Linux/macOS 安装 Rust
- rustup 版本管理器
- 镜像源配置（中国大陆用户）
- VS Code/IntelliJ IDEA 配置

#### 00.1. Cargo 教程
- 项目创建与管理
- 依赖管理（Cargo.toml）
- 构建与发布
- 工作空间（Workspace）
- 自定义命令

#### 00.2. 输出到命令行
- print! 和 println! 宏
- 格式化输出（{}, {:?}, {:#?}）
- 数值格式化（宽度、精度、进制）
- format! 宏
- eprint! 和 eprintln!

#### 00.3. 注释
- 行注释（//）
- 块注释（/* */）
- 文档注释（/// 和 //!）
- 文档生成（cargo doc）

#### 00.4. 运算符
- 算术运算符（+, -, *, /, %）
- 比较运算符（==, !=, <, >, <=, >=）
- 逻辑运算符（&&, ||, !）
- 位运算符（&, |, ^, <<, >>）
- 类型转换运算符（as）
- 运算符重载

### 第一阶段：基础语法（4章）

#### 01. 变量
- 变量声明与可变性 (`let`, `mut`)
- 变量遮蔽 (Variable Shadowing)

- 标量类型：整数、浮点数、布尔值、字符

#### 03. 函数
- 函数定义与参数
- 返回值与表达式
- 闭包 (Closure)
- 高阶函数

#### 04. 流程控制
- `if` 表达式
- `loop`、`while`、`for` 循环
- `match` 表达式
- `if let` 和 `while let`

### 第二阶段：核心概念（3章）

#### 05. 所有权
- 所有权三大规则
- 移动语义 (Move Semantics)
- 引用与借用
- 切片 (Slice)

#### 06. 结构体
- 结构体定义与实例化
- 方法定义 (`impl` 块)
- 关联函数

#### 07. 枚举
- 枚举定义与使用
- `Option<T>` 和 `Result<T, E>`
- 模式匹配

### 第三阶段：标准库与工程化（3章）

#### 08. 集合
- Vector：动态数组

#### 09. 模块
- 模块定义 (`mod`)
- 路径与 `use` 关键字
- `pub` 可见性控制

#### 10. 错误处理
- 不可恢复错误：`panic!`
- 可恢复错误：`Result<T, E>`

### 第四阶段：高级特性（3章）

#### 11. 泛型
- 泛型函数、结构体、枚举
- Trait Bound
- `where` 子句

#### 12. Trait
- Trait 定义与实现
- 默认实现
- 动态分发 (`dyn Trait`)

#### 13. 生命周期
- 生命周期注解语法
- 生命周期省略规则

### 第五阶段：实战进阶（7章）

#### 14. 迭代器
- Iterator trait
- 适配器方法（map, filter, take）
- 消费者方法（collect, fold, sum）
- 自定义迭代器

#### 15. 文件与IO
- 文件读写操作
- 标准输入输出
- 路径操作（Path, PathBuf）
- 缓冲IO（BufReader, BufWriter）

#### 16. 面向对象
- 封装（pub 控制）
- 继承模拟（Trait 继承）
- 多态（Trait 对象）
- 设计模式（工厂、策略、观察者、构建者）

#### 17. 并发编程
- 线程创建与管理
- 消息传递（Channel）
- 共享状态（Mutex, RwLock）
- Send 和 Sync Trait

#### 18. 宏
- 声明宏（macro_rules!）
- 过程宏（派生、属性、函数式）
- 内置宏（vec!, println!, dbg!）

#### 19. 智能指针
- Box：堆分配
- Rc：引用计数（单线程）
- Arc：原子引用计数（多线程）
- RefCell：内部可变性
- Mutex 和 RwLock

#### 20. 异步编程
- async/await 语法
- Future trait
- 异步运行时（Tokio, async-std）
- 异步IO和异步通道

## 快速开始

### 环境要求

- Rust 1.70 或更高版本
- Cargo（Rust 包管理器）

### 运行项目

```bash
# 克隆仓库
git clone git@github.com:Sogrey/Rust-Guide.git
cd Rust-Guide

# 运行所有示例
cargo run

# 编译检查
cargo check

# 构建项目
cargo build
```

### 运行特定示例

如果你想单独学习某个主题，可以在 `main.rs` 中注释掉其他模块，只保留你想学习的部分。

## 学习建议

### 学习路线

1. **初学者**：按顺序从第零阶段开始，先搭建环境，再学习基础语法
2. **有编程经验**：可以跳过环境搭建，直接从基础语法开始
3. **有其他语言经验**：重点关注所有权、生命周期等 Rust 特有概念

### 学习方法

1. **按顺序学习**：每个主题都是后续内容的基础，建议按顺序学习
2. **动手实践**：每个示例都可以直接运行，建议修改代码并观察结果
3. **阅读注释**：代码中包含详细的中文注释，帮助理解概念
4. **阅读文档**：每个章节都有配套的教程文档
5. **查阅官方文档**：遇到不理解的概念，可以查阅 [Rust 官方文档](https://doc.rust-lang.org/)
